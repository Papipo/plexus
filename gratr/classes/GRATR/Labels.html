cost(u,v=nil,weight=nil)

A generic cost function. It either calls the weight function with and edge constructed from the two nodes, or calls the [] operator of the label when given a value. If no weight value is specified, the label itself is treated as the cost value.

Note: This function will not work for Pseudo or Multi graphs at present. FIXME: Remove u,v interface to fix Pseudo Multi graph problems.
